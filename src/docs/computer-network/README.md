# 计算机网络面试题

本章节涵盖计算机网络相关的面试题，包括HTTP/HTTPS、TCP/IP、网络协议等核心知识点。

## 核心知识点

### 1. TCP/IP协议栈
- **TCP vs UDP**：连接导向 vs 无连接
- **三次握手与四次挥手**：建立连接与断开连接过程
- **流量控制与拥塞控制**：滑动窗口、慢启动、拥塞避免
- **TCP优化**：Nagle算法、延迟ACK、快速重传

### 2. HTTP协议
- **HTTP/1.1 vs HTTP/2 vs HTTP/3**：特性对比
- **HTTPS**：TLS握手过程、证书验证
- **RESTful API设计**：资源定义、状态码、幂等性
- **缓存机制**：Cache-Control、ETag、Last-Modified

### 3. 网络编程
- **Socket编程**：TCP/UDP Socket、select/poll/epoll
- **IO多路复用**：select、poll、epoll原理与对比
- **网络性能优化**：零拷贝、TCP参数调优

## 常见面试题

### 基础题
1. **Q：TCP三次握手的过程是什么？**
   A：客户端发送SYN报文，服务器回复SYN+ACK，客户端回复ACK...

2. **Q：HTTP和HTTPS的主要区别是什么？**
   A：HTTPS在HTTP基础上增加了TLS/SSL加密层...

### 进阶题
1. **Q：解释TCP流量控制和拥塞控制的区别**
   A：流量控制是端到端的，防止发送方发送过快...

2. **Q：HTTP/2相比HTTP/1.1有哪些改进？**
   A：二进制分帧、多路复用、头部压缩、服务器推送...
